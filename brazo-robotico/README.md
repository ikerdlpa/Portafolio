# Brazo robótico

## Descripción
Desarrollo de un brazo robótico funcional controlado mediante microcontrolador y servomotores.

## Objetivo
Simular movimiento mecánico controlado con precisión mediante integración de hardware y software.

## Tecnologías utilizadas
- ESP32 / Arduino  
- Servomotores  
- Fusion 360  
- Electrónica básica  

## Desarrollo
- Modelado de piezas en CAD  
- Ensamblaje mecánico del sistema  
- Conexión e integración de servomotores  
- Programación para control de movimiento  

## Retos y soluciones
- Inestabilidad en movimientos → ajuste de alimentación y control  
- Precisión limitada → calibración de ángulos  
- Problemas mecánicos → rediseño de piezas  

## Resultados
- Movimiento funcional del brazo  
- Control estable de los servos
