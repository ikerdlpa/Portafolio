# Monitor de pulsos con ESP32

## Descripción
Sistema de medición de frecuencia cardíaca en tiempo real utilizando un sensor de pulso y ESP32.

## Objetivo
Obtener lecturas de frecuencia cardíaca de forma accesible y funcional.

## Tecnologías utilizadas
- ESP32  
- Sensor de pulso  
- Arduino IDE  

## Desarrollo
- Lectura de datos del sensor  
- Procesamiento de señal  
- Filtrado de ruido  
- Visualización en monitor serial  

## Retos y soluciones
- Ruido en la señal → implementación de filtrado  
- Lecturas inestables → calibración del sistema  
- Sensibilidad del sensor → pruebas en diferentes condiciones
